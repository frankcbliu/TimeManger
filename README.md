## User Install（安装）

打开[release](https://github.com/frankcbliu/TimeManger/releases)页面，打开`Assets`列表后下载`.dmg`后缀文件到本地安装即可。


## Developer Install（开发者安装）

### Install

```sh
yarn install
```

### Usage

```sh
yarn run dev
```

### Build
```sh
yarn run build
```
